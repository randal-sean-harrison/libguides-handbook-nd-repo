Button

Buttons are set to open in a new window by default, though this setting can be changed. If using multiple buttons on your page, make sure that all the buttons are the same size. Button defaults are: Sized Normal, colored Primary(standard blue), and aligned Center.

Alternate Button Colors
Alternate button colors should only be used to signal an alternate mood. For example, a yellow button might sound a note of caution; a red button might signify probibition or danger.

Specialty Buttons

Buttons commonly invoke a call to other services, such as opening an email client, calendar booking application, or a modal image preview window, etc. For help with more advanced buttons, contact the LibGuides Management Team.
