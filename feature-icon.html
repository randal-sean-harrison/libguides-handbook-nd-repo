Specialty Classes — Rotation / Size / Spin

Spacing Classes

These classes add horizontal spacing around Font Awesome icons.

<i class="fa fa-gear fa-spacer-right" aria-hidden="true"></i>
Example CSS Class Description
Text Label -- Normal—no spacing
Text Label fa-spacer-right Significant right spacing
Text Label fa-spacer-left Significant left spacing
Text Label fa-fw Fixed-width spacing

Rotation Classes

These classes may be added to rotate Font Awesome icons.

<i class="fa fa-key fa-rotate-90" aria-hidden="true"></i>
Example CSS Class Description
-- Normal—no rotation
fa-rotate-90 Rotate icon 90°
fa-rotate-180 Rotate icon 180°
fa-rotate-270 Rotate icon 270°
fa-flip-horizontal Flip the icon horizontally
fa-flip-vertical Flip the icon vertically

Size Classes

You may specify an alternate size with the following classes.

<i class="fa fa-camera-retro fa-3x" aria-hidden="true"></i>
Example CSS Class Description
-- Standard sized
fa-lg Super-sized
fa-2x Double-sized
fa-3x Triple-sized
fa-4x Quadruple-sized
fa-5x Quintuple-sized

Spin Classes

These classes may be added to spin certain Font Awesome icons. Some browsers don't support this effect (e.g., IE8, IE9), or have issues rendering animated icons. We recommend using this technique sparingly, if at all.

<i class="fa fa-cog fa-spin fa-lg fa-fw"></i>
Example CSS Class Description
Loading... fa-spin Smooth spin on a standard fa-spinner icon
Loading... fa-pulse Stepwise spin on a standard fa-spinner icon
Loading... fa-spin Smooth spin on a large fa-circle-o-notch icon
Loading... fa-spin Smooth spin on a large fa-refresh icon
Loading... fa-spin Smooth spin on a large fa-cog icon
Loading... fa-spin Smooth spin on a large fa-compass icon
Loading... fa-spin Smooth spin on a large fa-plane icon *

* Obviously not all icons look amazing when spinning...
